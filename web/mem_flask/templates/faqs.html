Cómo empezar

Para hacer funcionar la herramienta se necesitan 4 archivos con nombres concretos:
  • Archivo JSON con los logs del día del examen: exam_logs.json
  • Archivo JSON con las respuestas de los estudiantes para cada pregunta: exam_answers.json
  • Archivo JSON con la puntuación de los estudiantes para cada pregunta: exam_califications.json
  • Archivo XML con el conjunto global de preguntas que el profesor ha usado para crear el examen en Moodle. exam_questions_set.xml
